Favorite places

*Add:
To add favorite points you must enter -Plane trip, point of origin- and there will be options to add "home" and "work" as a favorite When selecting the option you must move the pin on the map and complete the address. If you want to add more favorite points you must select Add move the pin on the map and give the name you want to the favorite point.

*Modify:
To modify a favorite point you must select the pencil to edit and you can modify the name of the favorite point and location on the map.

*Remove:
To delete one or more favorite places you must keep pressed and the option to select to remove will appear.
